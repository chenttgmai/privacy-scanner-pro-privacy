Privacy Scanner Pro

This policy page is localized for Bahasa Indonesia. The app performs OCR and document analysis on device by default and is designed for privacy first document handling.

1. Data Processing

Text recognition, ID card parsing, search, tagging, import and export features are intended to run locally on the user's iPhone or iPad. We do not require cloud upload for core functionality.

2. User Controlled Export

When you choose export, a JSON backup file is created only at the location you select. You may transfer it to another device that has this app installed by AirDrop, Files, or other local sharing tools.

3. Permissions

  • Photos permission is used only when you choose images from your library.
  • File access is used only for importing or exporting backups and text files.
